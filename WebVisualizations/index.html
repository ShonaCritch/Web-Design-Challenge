Summary: Latitude vs X


In this analysis project was undertaken to examine how weather changes in comparison with postion to the equator. This analysis was completed by sourcing data from the OpenWeatherMap API.

A random sample of over 500 cities was generated using CityPy and these cities formed the basis of the weather dataset called from the OpenWeatherMap API.
Using Matplotlib a number of visualisation of the latitude vs. weather were plotted to enable analysis of the weather vs. latitude.

In particular the weather factors plotted by latitude include: temperature, cloudiness, wind speed and humidity. This website provides the visualisations and analysis with descriptions of any trends and correlations observed.

The source data is available on the site and is provided for review, replication and futher use as required.
